# Tabela-de-pre-os
Após concluir a faculdade, iniciei um bootcamp de Desenvolvimento Web, onde venho adquirindo conhecimentos em diversas áreas e linguagens.
Nesta aula, desenvolvemos uma tabela de preços simples, com foco no uso de Flexbox, permitindo que o layout se adapte automaticamente de acordo com o dispositivo utilizado para acessar o site, garantindo uma melhor experiência responsiva.
